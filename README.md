# Super Smash Bros. Melee Randomizer

A brand new melee at the click of a button! SSBMr will apply a variety of mods via "flags" that will have an effect on the game. Mods include randomized hitboxes, throws, character attributes, music, textures and much more! Also features "balancing" which will keep the game competitive by keeping moves close to their original power level! SSBMr is fully compatible with real hardware and slippi so you can share the fun with friends!

## [Generate a seed on the Website by clicking here!](https://ssbmr.com)

### Multiplayer

Share the page link of the generated seed with the person you want to play with so that they can create the same randomized iso then you and your friend should both load that iso in Slippi and play like normal.

### Join the official [Discord](https://discord.gg/48SQfvcvVY) server!

### Command Line Usage

`randomizer.py "iso_path" "output_directory" "flags"`

# Features
- Flask front-end, styled with HTML, CSS, and JavaScript.
- Creates xdelta patches on server.
- Distributes xdelta patches and patches in browser.
- Python back-end, command line executable.
- Can replace .dat and .hps files allowing for customization.
- Rebuilds an ISO bootable on GameCube hardware.
- Active Discord community and weekly tournaments.
